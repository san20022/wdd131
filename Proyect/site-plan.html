Overview

Purpose

The purpose of this site is to educate, raise awareness, and inspire people to care for the environment through simple and sustainable actions. The site offers accessible information, practical tips, and useful resources that promote an eco-friendly lifestyle.

Audience

This site is designed for the general public—students, families, workers, and anyone interested in learning how to protect the planet. No prior knowledge of sustainability is needed, as the content is clear, practical, and easy to apply.

Dynamic elements

This site will include JavaScript-based dynamic elements such as: a random eco-friendly tip shown each time the page loads, a counter for eco-friendly actions completed by the user, and an interactive form where users can select sustainable habits. It will also show the current date and include a dark/light theme toggle.

Content

Home page

The home page will feature a welcome message, a brief introduction about the importance of protecting the environment, a section with recent eco-news, and a random eco tip displayed dynamically. A button to toggle between light and dark theme will also be included.

Tips

This page will show a dynamic list of environmental tips loaded from a JavaScript or JSON file. It will include a form where users can check off the tips they’ve practiced, and it will display a total count.
